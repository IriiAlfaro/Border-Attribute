Border Atribute

Línea que limita la parte exterior o da importancia a cierto contenido específico. La propiedad de CSS, Border es una propiedad abrevieda, para establecer los valores individuales del atributo en una sola línea

Syntax

        div {
            border: 'length' 'style' 'color';
        }

Border-Radius

La propiedad border-radius permite a los creadores de sitios web, definir que tan redondo es o son las equinas del borde del elemento. 
        .example-1 {
            background-color: #00DB60;
            border-radius: 50%;
            height: 100px;
            margin: 0.5em 0;
            width: 200px;
        }
        /* Esto se le aplica a las 4 esquinas */
  • Si se establece un valor, este radio se aplica a las cuatro esquinas.
  • Si se establecen dos valores, el primero se aplica a border-top-left y border-bottom-right, la segunda se aplica a la esquina border-top-right y border-bottom-left.
  • Cuatro valores que aplican a la parte border-top-left, border-top-right,border-bottom-right, border-bottom-left, en ese orden.
  • Tres valores: el segundo valor se aplica a border-top-right y también a border-bottom-left.
        .example-2 {
            background-color: #00DB60;
            border-radius: 12px 54px;
            display: inline-block;
            height: 8em;
            margin: 1em;
            width: 10%;
        }
        /* Esto se aplicaría a border-top-left//border-bottom-right & 
       border-top-right//border-bottom-left */

        .example-3 {
            border: 20px groove #EB464C;
            border-radius: 10px 40px 40px 10px;
            display: inline-block;
            height: 8em;
            margin: 1em;
            width: 10%;
        }
        /* Esto definiria el tamaño de cada esquina, top-left | top-right | 
       bottom-right | bottom-left */

        .example-4 {
            background-color: #E56928;
            border-radius: 4px 16px 9px;
            display: inline-block;
            margin: 1em;
            height: 8em;
            width: 10%;
        }
        /* El segundo valor se aplica a border-top-right y también a 
       border-bottom-left. */

También puede especificar los radios en el cual está rodeada la esquina. En otras palabras, el redondeo no tiene que ser perfectamente circular, puede ser elíptica. Esto se hace usando una barra diagonal ("/") entre dos valores.

        .example-5 {
            border-radius: 10px/30px; /* horizontal radius / vertical radius */
        }

        .example-6 {
            border-radius: 30px/10px; /* horizontal radius / vertical radius */
        }

Lo soporta:

  • IE 11
  • Firefox 36
  • Chrome 41
  • Safari 8
  • Opera 27

Triángulos

En CSS podemos definir cada uno de los cuatro bordes de un bloque de forma diferente. Se puede aprovechar esta característica para dibujar figuras geométricas, como el triángulo: Por ejemplo, si definimos un borde de una determinada anchura y a los bordes adyacentes (contiguo, cercano, próximo, lindante) le damos la mitad de anchura y un color diferente, obtenemos una forma triangular si al propio contenedor le damos una anchura y altura de cero. Por ejemplo:

Border-Image

La propiedad de CSS border-image permite utilizar una imágen para definir los bordes de los elementos.La propiedad border-image es usada en lugar de la sentencia border-style.

        .border-image { 
            border: 15px solid transparent;
            padding: 10px 20px;
            -moz-border-image:url("/files/4127/border.png") 30 30 repeat; /* Old Firefox */
            -webkit-border-image:url("/files/4127/border.png") 30 30 repeat;/* Safari */ 
            -o-border-image:url("/files/4127/border.png") 30 30 repeat; /* Opera */
            border-image:url("/files/4127/border.png") 30 30 repeat;
        }
            
        .round{ 
            border: 15px solid transparent;
            padding: 10px 20px;
            -moz-border-image:url("/files/4127/border.png") 30 30 round;/* Old Firefox */ 
            -webkit-border-image:url("/files/4127/border.png") 30 30 round;/* Safari */ 
            -o-border-image:url("/files/4127/border.png") 30 30 round;/* Opera */ 
            border-image:url("/files/4127/border.png") 30 30 round;
        }
            
        .stretch{ 
                border: 15px solid transparent;
                padding: 10px 20px;
                -moz-border-image:url("/files/4127/border.png") 30 30 stretch;/* Old Firefox */ 
                -webkit-border-image:url("/files/4127/border.png") 30 30 stretch;/* Safari */ 
                -o-border-image:url("/files/4127/border.png") 30 30 stretch;/* Opera */         
                border-image:url("/files/4127/border.png") 30 30 stretch;
        }

Dato importarte usa prefijos.

existe la opcion round es una variable de repeat, la cual distribuye las partes de tal manera que los extremos se conecten bién.

La opción stretch estira la imágen para rellenar todo el borde del área es variable.

  • Chrome 7
  • Internet Explorer 11
