Border Atribute

Línea que limita la parte exterior o da importancia a cierto contenido específico. La propiedad de CSS, Borderes una propiedad abrevieda, para establecer los valores individuales del atributo en una sola línea

Syntax

        div {
            border: 'length' 'style' 'color';
        }

Esto aplcaría a todos los lados del elemento, ya si se desea combinar estilos, tamaños y colores, se debe utilizar border- y su lado específico, como veremos más adelante.

Border-Radius

La propiedad border-radius permite a los creadores de sitios web, definir que tan redondo es o son las equinas del borde del elemento. 
        .example-1 {
            border: 1px solid green;
            border-radius: 12px;
        }
        /* Esto se le aplica a las 4 esquinas */
  • Si se establece un valor, este radio se aplica a las cuatro esquinas.
  • Si se establecen dos valores, el primero se aplica a border-top-left y border-bottom-right, la segunda se aplica a la esquina border-top-right y border-bottom-left.
  • Cuatro valores que aplican a la parte border-top-left, border-top-right, border-bottom-right, border-bottom-left, en ese orden.
  • Tres valores: el segundo valor se aplica a border-top-right y también a border-bottom-left.
        .example-2 {
            border-radius: 12px 54px;
        }
        /* Esto se aplicaría a border-top-left//border-bottom-right & 
       border-top-right//border-bottom-left */

        .example-3 {
            border: 1px groove black;
            border-radius: 16em 23em 8em 12em;
        }
        /* Esto definiria el tamaño de cada esquina, top-left | top-right | 
       bottom-right | bottom-left */

        .example-4 {
            border-radius:4px 16px 9px;
        }
        /* El segundo valor se aplica a border-top-right y también a 
       border-bottom-left. */

También puede especificar los radios en el cual está rodeada la esquina. En otras palabras, el redondeo no tiene que ser perfectamente circular, puede ser elíptica. Esto se hace usando una barra diagonal ("/") entre dos valores.

        .example-5 {
            border-radius: 10px/30px; /* horizontal radius / vertical radius */
        }

        .example-6 {
            border-radius: 30px/10px; /* horizontal radius / vertical radius */
        }

Border-Right

Es una propiedad abreviada que establece los valores de border-right-color, border-right-style y border-right-width. Estas propiedades describen el borde derecho del elementos.

Syntax

        div {
            border-right: 'length' 'style' 'color';
        }

Ejemplo

        .example-7 {
            border-radius: 10px/30px; /* horizontal radius / vertical radius */
        }

Border-Style

El border-style, propiedad de CSS, es una propiedad abreviada para establecer el estilo de línea de los cuatro lados de los bordes entre elementos.

  • Si se establece un valor, el estilo se aplicara a los cuatro lados.
  • Si se establecen dos valores, el primero se aplica a border-top y border-bottom, la segunda se aplica border-right y border-left.
  • Cuatro valores que aplican a border-top, border-right, border-bottom, border-left, en ese orden.
  • Tres valores: el segundo valor se aplica a border-right y también a border-left.

Syntax

        div {
            border-style: 'style' /* {1,4} */
        }

Border-Top

Es una propiedad abreviada que establece los valores de border-top-color, border-top-style y border-top-width. Estas propiedades describen el borde superior de los elementos.

Syntax

        div {
            border-top: 'lenght' 'style' 'color';
        }

Ejemplo

        .example-8 {
            border-top: 10px groove black; 
        }

Border-Width

La propiedad border-width establece el ancho del borde de una caja. Usar la propiedad abreviada de border es más conveniente.

Syntax

        div {
            border-width: 'lenght'; /* {1,4} Keywords: thin | medium | thick
        }

Ejemplos

        .example-9 {
            border-width: 10px 8px 6px; 
        }

        .example-10 {
            border-width: medium thin thin thick; 
        }
